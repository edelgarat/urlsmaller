# urlsmaller
<h1>Install:</h1>
<h3>in project path (cmd):</h3>

	npm i

<h3>start mongodb:</h3>
<h3>in mongodb path (cmd):</h3>
	
	mongo

or
	
	mongo --dbpath path/to/db
   
<h3>open new cmd in mongodb path:</h3>
		
	mongod
	use urlSmaller

<h3>in project path (cmd):</h3>
		
	node app
		
<h2>and visit localhost</h2>
